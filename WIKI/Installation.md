# Installation

Install via pip (recommended)
```bash
pip install csv-mapper
```

Install from source
```bash
git clone https://github.com/Paddy-AM/csv-mapper.git
cd csv-mapper
python -m venv .venv
source .venv/bin/activate  # or .venv\Scripts\activate on Windows
pip install -e .
```

Requirements
- Python 3.8+
- Optional: PyYAML if using YAML mapping files

Notes
- For development, see the Development page for installing dev dependencies and running tests.
