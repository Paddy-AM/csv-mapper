# csv-mapper

csv-mapper is a lightweight Python library and CLI tool for mapping CSV data to structured formats (JSON, Python objects) using reusable mapping configurations. It focuses on flexible field mapping, type coercion, validation rules and simple transformation pipelines.

Key features
- Declarative mapping configuration (YAML / JSON)
- CLI for quick CSV-to-JSON conversions
- Python API for programmatic usage and integration
- Validation and type coercion
- Extensible transformers
